What is MAAFlo?

A self-hosted, open-source ad management hub for marketing agencies and in-house teams.

📋

Campaign Planning

Create campaigns with goals, budgets, and date ranges. Assign channels and daily budgets per platform, all from one screen.

🚀

One-Click Push

Push a campaign to Google Ads, Meta, TikTok, LinkedIn, and more simultaneously. Each platform gets its own campaign ID stored in MAAFlo.

📊

Unified Analytics

Pull spend, impressions, clicks, conversions, and revenue into a single daily analytics table. Compare channels side-by-side on the same chart.

👥

Multi-Client

Manage multiple clients from one install. Each client has isolated channels, campaigns, and credentials — no data bleeds between accounts.

🔑

Per-Client Credentials

OAuth tokens and API keys are stored per client in the database — not in environment variables — so each client authenticates independently.

🔒

Self-Hosted

Run on your own server. Your client data, credentials, and analytics never leave your infrastructure. SQLite keeps setup minimal.

How It Works

From setup to live campaign in five steps.

Create a Client

Add your client (or your own brand) in the Clients section. Each client is fully isolated — its own channels, campaigns, and credentials.

Connect Channels

Go to Channels and connect the platforms you use. Enter your app credentials, complete OAuth, and select the ad account you want to manage.

Create a Campaign

Define the campaign name, goal (awareness, traffic, leads, conversions, sales), budget, and date range.

Add Channels & Budgets

Inside the campaign, add each platform you want to run on. Set a per-channel daily budget. Create ad creatives per channel.

Push & Track

Hit Push for each channel. MAAFlo creates the campaign on the platform and stores the external campaign ID. Analytics sync daily so you can track performance without leaving MAAFlo.

Channel Onboarding Notes

Key requirements before connecting each platform.

Google Ads

Requires a Google Cloud OAuth app and a Google Ads Developer Token.

  1. Create a project in Google Cloud Console
  2. Enable the Google Ads API
  3. Create OAuth 2.0 credentials (Web Application)
  4. Apply for a Developer Token in Google Ads Manager → Tools → API Center
  5. Enter Client ID, Client Secret, and Developer Token in MAAFlo → Channels

Meta Ads

Easiest path: generate a long-lived token from the Graph API Explorer.

  1. Go to developers.facebook.com → your app → Graph API Explorer
  2. Select your app, generate a User Token with ads_management and ads_read permissions
  3. Paste the token into MAAFlo → Channels → Meta Ads
  4. Enter your Ad Account ID (find it in Ads Manager after "act_")

LinkedIn Ads

Requires Marketing Developer Platform approval from LinkedIn (1–3 business days).

  1. Create an app at linkedin.com/developers/apps
  2. Link the app to a LinkedIn Company Page (required to see the Products tab)
  3. Products tab → Request access to Marketing Developer Platform
  4. After approval: add redirect URL, enter Client ID & Secret in MAAFlo
  5. Alternative: use OAuth Token Tools in the Developer Portal to generate a token directly and paste it in MAAFlo

TikTok Ads

Register as a TikTok for Business developer to get app credentials.

  1. Go to business-api.tiktok.com → Become a Developer
  2. My Apps → Create App → add Marketing API product
  3. Add http://localhost:3001/api/channels/tiktok/callback as redirect URL
  4. Copy App ID and App Secret into MAAFlo → Channels → TikTok Ads
  5. Click Connect and authorize your Ads Manager account
